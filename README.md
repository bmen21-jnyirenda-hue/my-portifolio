# ⛏️ Mining Engineering Interactive Portfolio

A modern, interactive portfolio website built with **Python**, **Streamlit**, and custom **CSS/HTML** to showcase academic milestones, GIS projects, and engineering competencies. 

It features a customized UI wrapped in an organic **Earth Copper** design theme.

---

## ✨ Key Features
- **Custom Navigation UI:** Native Streamlit arrows are replaced with an airtight **☰ Menu** and **✖ Close** layout button.
- **Dynamic Project Showcase:** Interactive accordion tables detailing geospatial analysis and field research.
- **Visual Skill Meters:** Clean progress indicators tracking industry-standard engineering software packages.
- **Connected Contact Form:** A fully functional HTML email form powered by FormSubmit.
